本站的一些事

1. 背景

20年的寒假，是大三的寒假。在学校划水摸鱼两年的我，突然意识到不能再迷茫下去，决定在寒假中恶补落下的课程。经过大半个寒假的学习，在寒假结束的最后几天，决定实操一个小系统用来练手，最终选定了个人博客系统，打算使用SpringBoot + Thymeleaf + SemanticUI来实现。然而，还是没有抓住寒假的尾巴，由于疫情的原因，在这个“寒假”迎来了网课。

就这样，当初信心满满的博客就这样搁置了，直到…

2. 博客的搭建

转眼间，大半个学期就这样过了，平时课程学习之余也进行着专业课的探索，争取给实习做好准备。于是，决定在20年5月28日开始搭建自己的第一个博客。

2.1 使用技术

Hexo + hexo-theme-butterfly + 一些魔改

在技术的选择上，没有选择拾起以前的项目，而是使用Hexo搭建一个静态博客。

正如Hexo的官网所说，Hexo是一个简单、高效且强大的博客框架，而我在使用时也深有体会，只需要一些环境，你就可以在一集电视剧的时间内搭建出一个简易的博客。这样的一个博客虽然能够使用，但毫无美观可言，因此我又选择了 hexo-theme-butterfly 作为Hexo的主题。

看中hexo-theme-butterfly的主要原因就是它的风格和我原来搁置的博客很像，当然，如果你也想搭建一个博客，但是不喜欢这个主题，可以去GitHub上搜索 hexo theme 即可，可以找到许多好看的主题。

最后，博客基本的美观解决了，但是功能依旧有点少，于是就使用了一些大佬的魔改加上自己的理解，就完成了现在这样所看的博客界面。

3. 关于“默烦”

COVID-19 还未出现在人们的视野里时，那时候的大学生还在教室上课，临近期末也可以在寝室通宵复习。

幸好马原还是稳稳地压线过了 😌
